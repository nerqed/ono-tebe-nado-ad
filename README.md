# Оно тебе надо

Проектная работа спринта — вёрстка лендинга аукциона **«Оно тебе надо»**
(аукцион вещей, в которые никто не верил). Свёрстано по макету: семантическая
разметка, грид-шапка и подвал, обложка с переиспользуемым оверлеем, флекс-сетка
карточек лотов и секция «Об аукционе».

## Стек

- HTML5 (семантические теги: `header`, `main`, `footer`, `nav`, `address`, `article`)
- CSS3 (Grid, Flexbox, фоновые изображения, кастомные шрифты Raleway)

## Структура

```
├── index.html
├── fonts/            — шрифты Raleway + fonts.css
├── images/           — обложка, карточки лотов, логотипы, иконки соцсетей
└── styles/
    ├── global.css    — сброс и общие стили (из заготовки)
    └── style.css     — локальные стили проекта
```
